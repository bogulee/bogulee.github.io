# Learning & Project Blog

개발 과정에서 학습한 내용과 직접 구현한 프로젝트를 기록하는 기술 블로그 저장소입니다.

단순히 사용 방법을 정리하는 데 그치지 않고 기술을 선택한 이유와 구현 과정에서 발생한 문제, 원인을 확인하고 해결한 과정을 남기는 것을 목표로 합니다.

## 주요 콘텐츠

### 학습 기록

- 새로 학습하거나 다시 정리한 기술 개념
- 도구와 구성 요소의 역할 및 동작 원리
- 실습 과정에서 확인한 명령어와 결과
- 헷갈렸던 내용과 추가로 학습할 부분

### 프로젝트

- 시작한 배경과 해결하려는 문제
- 기술 스택을 선택한 이유
- 단계별 구현 및 검증 과정
- 장애 재현과 트러블슈팅
- 프로젝트 결과, 한계와 개선 방향

## 안내

이 저장소의 글은 학습 및 프로젝트 진행에 따라 계속 수정될 수 있습니다. 프로젝트의 실행 가능한 코드와 상세 설정은 각 프로젝트 저장소에서 관리합니다.
